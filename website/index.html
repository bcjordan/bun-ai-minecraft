Get Started

Installation

Clone the repository and install dependencies:

git clone https://github.com/yourusername/opencode-minecraft-bot.git
cd opencode-minecraft-bot
bun install

Configuration

Set up your environment variables:

cp .env.example .env

Run

bun run start
